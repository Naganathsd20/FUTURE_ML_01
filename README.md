FUTURE_ML_01

This repository contains my Machine Learning internship tasks completed as part of Future Interns.

Task 1 – Sales and Demand Forecasting
In this task, I worked on predicting future sales using historical data. I cleaned the dataset, created time-based features like month and year, and built a regression model. I also visualized the results to understand trends and predictions.

Task 2 – Ticket Classification and Prioritization
In this task, I built a model to classify support tickets and predict their priority. I used text preprocessing techniques and trained a Logistic Regression model. The system helps in automating customer support handling.

Tools and Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib

Overall, these projects helped me understand how Machine Learning can be applied to real business problems like sales forecasting and support automation.
